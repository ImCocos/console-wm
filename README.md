# Install

```bash
git clone https://github.com/ImCocos/console-wm.git
cd console-wm
python3.11 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

# Run

## Linux

```bash
cd console-wm
deactivate
source venv/bin/activate
sudo python main.py
```
